Motion Sensor using Passive Infrared

Arduino

The Motion Detector using PIR is an efficient and cost-effective security solution that detects motion in its surroundings. Powered by a PIR sensor, this system detects infrared radiation emitted by objects, such as humans or animals, that move within its range. The motion detected triggers an output signal, which can be used to activate alarms, lights, or other devices. This technology is widely used in security systems, automated lighting, and home automation, offering a simple yet reliable way to monitor movement in a given area.
